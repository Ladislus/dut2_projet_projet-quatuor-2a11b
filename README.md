# INSTALLATION

1 - Se placer dans le dossier Ressources/
2 - Exécuter setup.sh (./setup.sh)
3 - Définir le fichier / dossier application dans .flaskenv (FLASK_APP = <Application>)


# LANCEMENT DU PROJET

1 - Lancer la commande "flask init-db"
2 - Lancer la commande "flask new-admin <nomutilisateur> <motdepasse>"
3 - Lancer la commande "flask run"

# ACCES AU SITE

1 - Se rendre à l'adresse http://localhost:5000
